<b>Hover to Play:</b>

It sounds so simple, yet feels like a completey new experience.

Hovely aims to be the fastest, most efficient way to interact with mass media. Hover over a media box to play its content, create pages of your favorite youtube and soundcloud links, rate media to receive personalized recommendations, and follow the live feed to see what content is being added by other users in real time.
